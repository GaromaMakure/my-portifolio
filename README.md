# Garoma Makure — Portfolio

Professional portfolio website for **Garoma Makure**, Software Engineer graduate from Haramaya University.

## Tech Stack

- React + Vite
- Tailwind CSS
- Three.js / React Three Fiber
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Environment Variables

Copy `.env.example` to `.env` and add your EmailJS credentials for the contact form:

```bash
cp .env.example .env
```

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy

This project can be deployed to GitHub Pages, Vercel, or Netlify.
